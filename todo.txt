# Requisitos Funcionais

- O usuário deve ser capaz de fornecer um texto e obter um set de perguntas e respostas gerado
    a partir do texto fornecido. 
- O usuário deve ser capaz de fornecer o set de perguntas e respostas sem a necessidade um texto de apoio.
- O usuário deve ser capaz de selecionar se as perguntas no script serão executadas pelo robô Eva
    de forma aleatória.
- O usuário deve ser capaz de selecionar se as respostas para as perguntas serão sempre exatas ou se irão ser uma
    substring da resposta correta.
- O usuário deve ser capaz de selecionar se as perguntas terão múltiplas respostas possíveis OU apenas uma única resposta
- O usuário deve ser capaz de gerar um script evaML a partir dessas perguntas e respostas, bem como os extras acima descritos.

Extras:
    - O usuário deve poder gerar um JSON eva-equivalent a partir do script evaML (a ser utilizado no robô)
